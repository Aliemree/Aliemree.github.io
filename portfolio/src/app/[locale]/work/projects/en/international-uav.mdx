## Project Details

Developed an international-class unmanned aerial vehicle (UAV) with advanced autonomous capabilities. The project focused on creating a lightweight, efficient aircraft capable of performing complex missions autonomously.

## Key Features

- **Aircraft Design**: Designed a 150 cm wingspan UAV weighing under 4 kg with optimized rudder control for enhanced maneuverability.
- **Aerodynamic Simulations**: Conducted comprehensive aerodynamic and structural simulations using ANSYS CFD to optimize flight performance.
- **Autonomous Navigation**: Integrated ArduPilot flight controller for waypoint navigation and mission planning capabilities.
- **Computer Vision Integration**: Implemented Jetson Nano for onboard AI inference enabling real-time computer-vision-based task execution.

## Technologies Used

- **ArduPilot** - Open-source autopilot system for autonomous flight control
- **ANSYS CFD** - Computational fluid dynamics for aerodynamic analysis
- **Jetson Nano** - NVIDIA edge computing platform for AI inference
- **Python** - For computer vision algorithms and mission scripting

## Challenges and Learnings

Balancing weight constraints with computational requirements was a significant challenge. The integration of Jetson Nano required careful power management and thermal considerations. Developing reliable failsafe mechanisms for autonomous operations enhanced understanding of safety-critical systems.
