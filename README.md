# Ali Emre - Kişisel Web Sayfası

Bu repo, Ali Emre'nin kişisel portfolyo web sitesini içerir. Site, HTML, CSS ve JavaScript kullanılarak geliştirilmiştir.

## İçerik

- **Ana Sayfa**: Kısa tanıtım ve uzmanlık alanları
- **Hakkımda**: Detaylı özgeçmiş, eğitim ve deneyimler
- **Projelerim**: TÜBİTAK, TEKNOFEST ve kişisel projeler
- **İletişim**: İletişim formu ve bilgileri

## Teknolojiler

- HTML5
- CSS3
- JavaScript
- Font Awesome ikonları
- Responsive tasarım

## Projeler

Sitede çeşitli projeler sergilenmektedir:
- İHA Tasarımı ve Simülasyonları (TEKNOFEST)
- Otonom Hedefleme Taret Sistemi (TEKNOFEST)
- Toplu Taşıma Yapay Zeka Sistemi (TÜBİTAK)
- 5G Tabanlı Konumlandırma Sistemi (TEKNOFEST)
- Finans Yönetim Sistemi (Kişisel)
- NLP Projeleri (TEKNOFEST)
- EYS - Engelli Yardım Sistemi (TEKNOFEST)
- İnekWiz - Akıllı Çiftlik İzleme (TÜBİTAK)
